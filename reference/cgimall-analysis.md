## 1. 인증 시스템

### 1-1. 성인인증 게이트
- 비바톤(Bbaton) 성인인증 — 사이트 접근 시 최초 1회 필수
- 미인증 시 JS alert + redirect로 전체 차단

### 1-2. 회원 유형 2분기
| 구분 | 업소(구인)회원 | 개인(구직)회원 |
|------|---------------|---------------|
| 주요 기능 | 채용정보 등록/관리, 인재관리, 맞춤인재 설정, 문의 | 이력서 등록/수정, 지원내역, 맞춤알림, 문의 |

### 1-3. SNS 로그인 4종
- Facebook, Google, Kakao Talk, Naver

### 1-4. 회원가입 필드
- 주민등록번호 앞6 + 뒤7 (성인 검증, 레거시)
- 비바톤 인증 (현대)
